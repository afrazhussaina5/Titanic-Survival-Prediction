# Titanic-Survival-Prediction
Titanic Survival Prediction using Machine Learning and Data Science techniques. This project includes data preprocessing, exploratory data analysis (EDA), feature engineering, visualization, and predictive modeling using Python, Pandas, NumPy, Scikit-Learn, and Google Colab.

## Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.
